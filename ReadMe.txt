arduino.py
Класс для отправления микроконтроллеру управляющих команд от бортового компьютера.
Код предоставлен для ознакомления с управляющими командами, которые может отправлять одноплатный компьютер.

runner.py
Программа, запускающаяся на одноплатном компьютере, для движения по разметке.

runner_detection.py
Программа, запускающаяся на одноплатном компьютере, для движения по разметке и детектирования объектов.
В программе содерится пример инференса нейросетевого детектора.

utils.py
Файл с функциями для обработки видеопотока: бинаризации, поиска линий разметки и т.п.
Используется программами runner, runner_detection.

classes.txt
Файл с описанием классов для нейросетевого детектора.
Используется программой runner_detection.

yolov4-tiny_uint8.tmfile
квантованная модель нейросетевого детектора.
Используется программой runner_detection.

quant_tool.py
программа для квантования модели нейронной сети
ВНИМАНИЕ! На данный момент поддерживаются только следующие модели детекторов:
Yolov3, Yolov3-tiny, Yolov4, Yolov4-tiny.
После обучения нейросетевого детектора, вы получите файл весов и файл конфигурации модели.
Необходимо подготовить их и набор из 500-1000 изображений с объектами для детектора.
Для получения квантованой модели запустите программу следующим образом:
python3 quant_tool.py -w <путь_до_весов> -c <путь_до_конфига> -d <путь_до_датасета>
После выполнения скрипта рядом с весами исходной модели появится .tmfile. Это и есть квантованая модель детектора.
